# go语法
## 标识符使用
- 用作空变量：
``` go
  _ = x
```
- 包导入使用:
``` go
  import _ xxx
```

## 空接口
```go
  interface{}
```
任何类型都是空接口类型的实现类型

## ++和--是语句不是表达式
```go
*p-- 等同于 (*p)--
```

## 表达式
- 表达式类型
```
选择表达式 context.Speaker
索引表达式 array[1]
切片表达式 slice[1:2]
类型断言 v1.(I1)
调用表达式 v1.M1()
```
- 断言表达式
  v1.(I1)
  - 如果v1是非接口值，这需要先转化成接口值，
    interface{}(v1).(I1)
  - 类型断言失败会引发运行时恐慌，使用如下
    var i1, ok = interface{}(v1).(I1)

### 基本类型
- 有18种基础类型
```
rune类型是专门用来存储Unicode编码的单个字符
```
rune有5种表示形式

- 字符串字面量有两种表示形式
  - 原生字符串字面量（\`）：所见即所得
  - 解释型字符串字面量（"）：解释转义字符

- 字符串的值不可变

- 常量只能使用基础类型，常量需使用const
